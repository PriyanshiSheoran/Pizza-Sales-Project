Pizza Sales SQL Analysis
Overview

This project presents a structured SQL analysis of pizza sales data to generate meaningful business insights. It demonstrates the use of relational database concepts and query techniques to evaluate sales performance and customer purchasing behavior.

Database Schema

The analysis is based on three relational tables:

orders – stores order date and time
order_details – contains order-level quantity details
pizza_types – includes pizza name, category, and size
Objectives
Calculate total revenue and order volume
Identify top-performing pizzas and categories
Analyze sales trends over time
