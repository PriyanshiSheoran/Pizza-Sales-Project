🍕 Pizza Sales SQL Analysis
📌 Overview

This project performs SQL-based analysis on pizza sales data to extract meaningful business insights. It demonstrates the use of relational databases to analyze sales performance and customer trends.

🗂️ Database Schema

The dataset consists of three tables:

orders – stores order date and time
order_details – contains order-wise pizza quantities
pizza_types – includes pizza name, category, and size
🎯 Objectives
Analyze total revenue and order volume
Identify top-performing pizzas
Evaluate sales trends over time
🛠️ Tech Stack
SQL (MySQL / PostgreSQL)
🚀 Conclusion

This project highlights how SQL can be effectively used to derive actionable insights from structured sales data.
