🍕 Pizza Sales SQL Analysis
📌 Overview

This project presents a comprehensive SQL-based analysis of pizza sales data, aimed at deriving actionable insights into business performance and customer purchasing patterns. It demonstrates the effective use of relational database concepts and query techniques.

🗂️ Dataset

The dataset is structured across three relational tables:

orders – captures order identifiers along with date and time
order_details – records item-level details including quantity per order
pizza_types – defines pizza attributes such as name, category, and size
🎯 Objectives
Evaluate overall revenue and order volume
Identify top-performing products and categories
Analyze sales trends across different time periods
🛠️ SQL Techniques
Table Joins
Aggregation Functions (SUM, COUNT)
GROUP BY Clauses
Subqueries
💻 Tools & Technologies
MySQL / PostgreSQL
🚀 Conclusion

This project highlights the application of SQL in transforming raw transactional data into meaningful insights, enabling data-driven decision-making and performance evaluation.
